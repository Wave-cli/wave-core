# version 0.2.2

## features

- **Colored error output**: Plugin and core errors displayed as `code: message` in red for better visibility
- **User-friendly messages**: Simplified error format for end users
- **Debug mode via env var**: Set `WAVE_DEBUG=1` to see full JSON error structure for plugin and core errors
- **Install script**: Automated setup script for easy installation to `~/.local/bin`
- **Update script**: Build and setup script for wave and flow plugins
- **AGENTS.md**: Documentation for AI-assisted development workflows
- **WAVE_DEBUG**: SDK error package now checks `WAVE_DEBUG` env var instead of accepting debug parameter

## fixes

- **Clean build in update.sh**: Added `go clean` before each build to ensure fresh builds
- **Clean version output**: Version now shows `v0.2.2` instead of `v0.2.2-5-g7ba18fb`
- **Core errors honor WAVE_DEBUG**: `wave <unknown>` now uses wave error formatting and honors `WAVE_DEBUG=1`
