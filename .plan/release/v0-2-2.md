# version 0.2.2

## features

- **Colored error output**: Errors displayed as `code: message` in red for better visibility
- **User-friendly messages**: Simplified error format for end users
- **Debug mode via env var**: Set `WAVE_DEBUG=1` to see full JSON error structure
- **Install script**: Automated setup script for easy installation to `~/.local/bin`
- **Update script**: Build and setup script for wave and flow plugins
- **AGENTS.md**: Documentation for AI-assisted development workflows

## fixes

- Replaced `--debug` CLI flag with `WAVE_DEBUG=1` environment variable for debug output
- SDK error package now checks `WAVE_DEBUG` env var instead of accepting debug parameter
